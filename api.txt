Create : 
Windows :
curl -X POST "http://localhost:8080/users/create" ^
-d "pseudo=Bob" ^
-d "email=bob@mail.com" ^
-d "password=1234"
Autre : 
curl -X POST http://localhost:8080/users/create \
-d "pseudo=Bob" \
-d "email=bob@mail.com" \
-d "password=1234"


Login :
Windows : 
curl -X POST "http://localhost:8080/users/login" ^
-d "email=bob@mail.com" ^
-d "password=1234"


Autre : 
curl -X POST http://localhost:8080/users/login \
-d "email=bob@mail.com" \
-d "password=1234"
